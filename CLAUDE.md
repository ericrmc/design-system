## Commit workflow

Before finishing your response, stage all changed/new files, commit with a concise message, and push. Use git directly — no intermediate files.

## Multi-agent work

You also have access to the Google `gemini` and OpenAI `codex` CLI tools for non-Anthropic LLM access. Codex is preferred for any thinking or reasoning tasks.
